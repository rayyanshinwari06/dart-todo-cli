# Dart CLI To-Do App 📝

A simple command-line to-do list application built in Dart.  
Add and view tasks directly from your terminal.

## ✨ Features
- Add new tasks
- View your task list
- Delete task
- Exit the app cleanly

## 🛠 How to Run

Make sure Dart SDK is installed, then use:

```bash
dart run bin/dart.dart
